# PWA 홈 화면 추가 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Android/iOS 홈 화면에 앱을 추가할 수 있도록 Web App Manifest, 앱 아이콘, 최소 서비스 워커를 구현한다.

**Architecture:** Next.js App Router 내장 API만 사용한다. `app/manifest.js`로 Web App Manifest를 선언하고, `app/icon.js` / `app/apple-icon.js`로 PNG 아이콘을 동적 생성한다. `public/sw.js`에 최소 서비스 워커를 두고 `app/providers.jsx`에서 등록한다. `app/layout.js`에 테마 컬러 메타를 추가한다.

**Tech Stack:** next/og (ImageResponse), Next.js App Router metadata API, 포트 3020

---

## 파일 구조

```
app/manifest.js        # Create: Web App Manifest
app/icon.js            # Create: 512×512 앱 아이콘
app/apple-icon.js      # Create: 180×180 iOS 아이콘
public/sw.js           # Create: 최소 서비스 워커
app/providers.jsx      # Modify: SW 등록 useEffect 추가
app/layout.js          # Modify: themeColor + appleWebApp 메타 추가
```

import { SessionProvider } from 'next-auth/react'

  useEffect(() => {
